Se houver mudanças em uma tabela do seu banco de dados (via SQLAlchemy ou comandos SQL), não será necessário refazer todo o processo de construção (via SQLAlchemy ou comandos SQL).
Basta usar a extensão `flask-migrate` para automatizar esse processo.

1) Para inicializar o migrate (ou seja, cria a pasta `migrations` caso ela não exista), escreva:
flask db init

2) Se existir a pasta `migrations`, digite:
flask db migrate -m "Initial migrate"  :: Parecido com o git

3) Por fim, para atualizar, faça:
flask db upgrate
